## 🛠️ How to Use

1. **Clone this repository**:
   ```bash
   git clone https://github.com/raihanuldev/Algorithms.git
   cd Algorithms
   ```

2. **Compile any algorithm file using g++**:
   ```bash
   g++ path/to/your/code.cpp -o output
   ./output
   ```

3. **Explore, modify, and practice!**
